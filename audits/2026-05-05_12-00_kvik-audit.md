## FEATURES IMPLEMENTIRANE
- Kompletan osnovni tok obrta: auth, onboarding, postavke, dashboard.
- Racuni i ponude (multi-item, popusti, dostava, email, PDF, statusi, storno, recurring flag).
- KPR i PO-SD alati + PDF/XLSX export.
- Share/referral sustav:
  - javni AI share linkovi (`/share/[uuid]`)
  - referral redirect (`/r/[code]`)
  - API pracenje aktivacija/posjeta i dashboard summary.
- Fiskalizacija 1.0 (implementirano u kodu):
  - enkripcija certifikata i lozinke (`src/lib/fiscalization/encryption.ts`)
  - validacija `.p12` (`certificate.ts`)
  - ZKI generator (`zki.ts`)
  - CIS SOAP klijent (`cis.ts`)
  - orchestrator (`fiscalize.ts`)
  - certifikat management API + UI wizard
  - fiskal status u listi racuna i JIR/ZKI u PDF-u kad postoje.

## FEATURES DJELOMICNO IMPLEMENTIRANE
- Fiskalizacija 1.0: backend i UI su implementirani, ali produkcijska potvrda nije gotova (nema verificiran DEMO end-to-end testa).
- CIS WS-Security u `cis.ts` je pojednostavljen i nije pun produkcijski XML-DSig flow.
- Recurring racuni: cuva se konfiguracija, nema automatiziranog scheduler-a.
- eRacuni (F2.0): jos nema izdavanja/zaprimanja.

## FEATURES KOJE NEDOSTAJU
